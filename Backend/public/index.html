Snap-Med API

Just to give you a rough idea.

Available Endpoints:

POST /api/auth/register - Register a new user with phone number and password

POST /api/auth/login - Authenticate a user and receive session cookies

GET /api/auth/session - Check if the current session is valid

POST /api/auth/logout - End the current session

POST /api/history - Save medication history (requires auth)

GET /api/history - Retrieve user's medication history (requires auth)

For more information, Contact the Owner.
